# CitiesApp - .NET 8 Web API (EF Core + Repository + Service)
Minimal project scaffolding to demonstrate SOLID principles.
To run:
1. Open the CitiesApp.Api folder with VS 2022/2023 or `dotnet` CLI.
2. `dotnet restore`
3. `dotnet ef database update` (configure connection string if needed)
4. `dotnet run`
